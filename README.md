# Система мониторинга уровня загруженности ПК

Система мониторинга уровня загруженности ПК предназначена для отслеживания использования ресурсов компьютера, включая процессор, оперативную память и жесткий диск. Программа отображает текущие показатели в реальном времени и предоставляет возможность записи данных в базу данных для последующего анализа.

## Установка

1. Установите Python версии 3.9.5 или выше.
2. Установите зависимости, указанные в файле requirements.txt:
   
   pip install -r requirements.txt

3. Создайте базу данных SQLite, выполнив скрипт создания базы данных:
   
   python system_data.py
   

## Запуск программы

1. Запустите программу:
   
   python tz.py
   

2. Откроется главное окно программы, где вы сможете наблюдать за показателями нагрузки на систему (ЦП, ОЗУ, ПЗУ).

3. Нажмите кнопку "Начать запись", чтобы начать запись данных в базу данных.

4. Когда закончите запись, нажмите кнопку "Остановить".
   

## Инструкция по использованию

### Главное окно

Главное окно программы содержит три секции:

- Загрузка процессора (ЦП)
- Свободное количество оперативной памяти (ОЗУ)
- Свободное пространство на жестком диске (ПЗУ)

Каждые n секунд (значение n задаётся в настройках программы) эти показатели обновляются.

### Запись данных

Нажав кнопку "Начать запись", программа начнёт сохранять данные в базу данных. Кнопка "Остановить" остановит запись.

### Просмотр сохранённых данных

Для просмотра сохранённых данных используйте любой инструмент для работы с базами данных SQLite, например, SQLite Browser.
